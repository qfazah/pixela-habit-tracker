# 📊 Code Pulse: Pixela Habit Tracker

A Python application designed to help developers track their daily coding habits. By integrating with the **Pixela API**, this script automates the creation of a visual, GitHub-style intensity graph to track hours spent coding over time.

---

## 🚀 Features

* **Automated User Registration:** Handles secure Pixela user creation.
* **Custom Graph Generation:** Creates a personalized tracking graph (`shibafu` green) measured in float hours.
* **Precision Logging:** Formats and posts exact time data to specific dates seamlessly.
* **Secure Coding Practices:** Implements environment variables to protect API tokens.

## 🛠️ Tech Stack

* **Language:** Python 3.x
* **Libraries:** `requests` for API communication, `datetime` for timestamp handling.
* **API Platform:** [Pixela API](https://pixe.la/)
